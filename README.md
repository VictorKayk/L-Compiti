## Getting start

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/VictorKayk/L-Compitti.git
   ```
2. Install packages
   ```sh
   yarn install
   ```
3. Build the project
   ```sh
   yarn build
   ```
4. Active the developer extension mode on your Chrome browser.
5. Load dist folder on extension page in your Chrome browser.
6. Enjoy :).
